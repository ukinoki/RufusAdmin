Pour utiliser linuxdeployqt
Dans un dossier perso genre home/rintintin/test
Créer l'arborisation
└── usr
    ├── bin
    │   └── RufusAdminApp
    └── share
        ├── applications
        │   └── rufusadmin.desktop
        └── icons
            └── hicolor
                └── 256x256 
                    └── apps 
                        └── rufusadmin.png
Coller dans le sous-dossier /bin le RufusAdminApp qui vient d'être compilé par Qt
Puis, lancer la commande
~/linuxdeployqt(nom de la version installée) chemin-vers-usr/usr/share/applications/RufusAdmin.desktop -appimage

Pour créer une appimage RufusAdmin automatiquement
	copier 
		le script create-rufusadmin-appimage.sh
		le script install-rufusadmin.sh
		le fichier RufusAdminQt6App
		le programme linuxdeployqt
	dans le dossier utilisateur
Clic droit sur les 2 scripts, onglet permissions, cocher autoriser l'éxécution comme un programme	
	
lancer le script create-RufusAdmin-appimage.sh
dans le terminal, entrer ./create-rufusadmin-appimage.sh
	le script va créer l'arborisation de l'appimage, puis l'appimage, puis détruira les fichiers temporaires
	
Pour installer RufusAdmin sur un poste à partir de l'appimage produite,
	lancer le script install-rufusadmin.sh
	dans le terminal, entrer ./install-eufusadmin.sh
